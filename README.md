### Tutorial Java 147
Java Learn 提供分享學習程式語言的成長與經驗，Java 是一種物件導向程式設計語言，但不是純物件導向的語言，因為支援基本類型，如 int 、 char 等，程式碼會編譯成 byte code ，運行在負責抽象底層操作系統的 JVM（Java 虛擬機）上，語法類似於 C/C++，但是不提供像 pointers 指針這樣的低階編程功能，擁有跨平台、物件導向、泛型程式設計的特性，廣泛應用於各種系統，如移動應用程式（ Android 基於 Java ）、桌面應用程式、Web 應用程式、客戶端-服務器應用程式、企業應用程式等。
